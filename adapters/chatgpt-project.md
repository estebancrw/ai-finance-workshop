# Adapter — ChatGPT (Projects)

## Setup (~5 minutes)

1. Go to chatgpt.com → sidebar → **New project** → name it `My Money KB`.
2. Open the project → **Instructions** → paste the full contents of
   `engine/project-instructions.md` (everything below the `---` line).
3. Add your cartridge:
   - **Preferred:** save your filled `snapshot.md` and `rules.md` as files
     and upload them to the project's **Files**.
   - **Fallback (no file upload on your tier):** start the first chat in
     the project with one message: `Here is my KB. Treat this as
     snapshot.md and rules.md:` followed by both files' contents pasted in.
4. Test with: `Where does my next peso go?` — the answer must cite *your*
   numbers and *your* rules. If it's generic, reply: `Check my KB first.`

## Refresh ritual (Level 2)

Monthly: edit `snapshot.md`, bump `as_of`, re-upload (or re-paste). Keep the
old snapshots — that's your history for trend questions.

## Notes

- Free tier has limited Projects support and message caps; Plus removes
  most friction.
- Memory (Settings → Personalization) is a bonus, not a substitute: the KB
  is explicit and yours; memory is implicit and the model's.
